# Smart Patch

Smart Patch is a low-cost AI-assisted crop advisory system that helps farmers detect early signs of tomato leaf diseases using a smartphone photo.

## Current Status
Prototype under development for hackathon submission.

## Tech Stack
- Flutter (Frontend)
- Supabase (Storage + Database)
- Python FastAPI (Backend)
- TensorFlow / PyTorch (ML Model)

## Note
Training data is sourced from publicly available agricultural datasets and is not redistributed.
